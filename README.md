# GestureCatcher
GestureCatcher is a tool based on LeapMotion. It analyzes underly data from LeapMotion and recognizes customized complex gestures.

## Tools
* XCode 7.0
* CocoaPods

## Install Steps
* check out the repo
* pod install
* open GestureCatcher.xcworkspace
